# 项目初始化说明

## 项目概述

本项目是一个现代化的电商平台，包含商品列表页和商品详情页两个核心页面。

## 技术栈

- **前端框架**: React 18.x + TypeScript
- **构建工具**: Vite
- **UI 样式**: Tailwind CSS
- **状态管理**: Zustand
- **路由**: React Router v6
- **数据模拟**: Mock.js
- **工具库**: 
  - clsx (动态类名)
  - lodash.debounce (防抖)

## 已安装的核心依赖

### 生产依赖
- `react` - React 核心库
- `react-dom` - React DOM 渲染
- `react-router-dom` - 路由管理
- `zustand` - 轻量级状态管理
- `mockjs` - 数据模拟
- `clsx` - 动态类名工具
- `lodash.debounce` - 防抖函数

### 开发依赖
- `typescript` - TypeScript 支持
- `vite` - 构建工具
- `@vitejs/plugin-react` - Vite React 插件
- `tailwindcss` - CSS 框架
- `postcss` - CSS 处理器
- `autoprefixer` - CSS 自动前缀
- `@types/*` - TypeScript 类型定义

## 路径别名配置

项目已配置 `@/` 作为 `src/` 的别名，可以使用以下方式导入：

```typescript
// 使用别名
import { Product } from '@/types'
import { useCart } from '@/hooks/useCart'

// 而不是相对路径
import { Product } from '../../types'
import { useCart } from '../../hooks/useCart'
```

## 配置文件说明

### vite.config.ts
- 配置了 React 插件
- 配置了路径别名 `@/` 指向 `src/`

### tsconfig.json
- TypeScript 编译配置
- 启用严格模式
- 配置路径别名映射

### tailwind.config.js
- Tailwind CSS 配置
- 指定内容扫描路径

### postcss.config.js
- PostCSS 配置
- 集成 Tailwind CSS 和 Autoprefixer
